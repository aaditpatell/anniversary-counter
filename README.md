# anniversary-counter 

https://bigoaktree.vercel.app/ 

Anniversary countdown for my partner (personal project & fun little thing to help me practice HTML)



Features

Live countdown timer (days, hours, minutes, seconds)
Progress bar tracking the journey to the 1-year mark
Rotating love phrases/inside jokes
Floating hand-drawn hearts with gentle drift animations
Floating photo gallery with physics-based movement and card-avoidance logic
Twinkling sparkle effects for aesthetics
Fully responsive design, viewable on computer & phone. 

Tech
Single-file HTML/CSS/JS. No dependencies, no build step. Just open index.html in a browser
or via the link https://bigoaktree.vercel.app/
